# Recommendation_using_NumPy
A NumPy based movie recommendation system that analyzes ratings and suggests unseen movies.

# Features
- Finds highest rated movie
- Finds lowest rated movie
- Recommends unseen movies
- Uses NumPy matrix operations

# Concepts Used
- NumPy arrays
- flatten()
- np.mean()
- np.argmax()
- np.argmin()
- np.where()
- axis operations
- Boolean masking
